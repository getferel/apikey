# API Key Tester

Simple API testing tool for major AI providers.

## Features

- **Provider Dropdown** - Anthropic, Gemini, OpenAI
- **API Key Input** - Secure password field
- **Query Input** - Type your prompt
- **Live Results** - See responses instantly
- **Quick Tests** - Pre-filled example prompts

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

```bash
vercel --prod
```

## Supported Providers

- Anthropic (Claude 3.5 Sonnet)
- Google Gemini (Gemini 2.0 Flash)
- OpenAI (GPT-4o)
